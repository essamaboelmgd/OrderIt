# OrderIt

## Project info

OrderIt is a modern web application built with React and Vite.

## Getting Started

To get started with development:

1. Clone the repository
2. Install dependencies:
   ```sh
   npm install
   ```
3. Start the development server:
   ```sh
   npm run dev
   ```

## Technologies

- Vite
- TypeScript
- React
- shadcn-ui
- Tailwind CSS
